Description:
------------
	
This module provides visitors to your web site with currency exchange rates.

Requirements:
-------------
	
This module requires drupal 4.5.x or greater.

Installation:
-------------

1. Copy the currency.module to the Drupal modules/ directory.

2. Allow access to the module under admin/user/configure/permission (4.5) or
   admin/access (4.6).
   You can enable it for authenticated and/or anonymous users, depending on
   your web site needs.

3. Configure the module under admin/settings/currency
   - Specify the default From Currency and the To Currency. You must enter the 
     three letter code for each. What you enter depends on your country and/or
     audience.

   - Provide a description. It can be any text you want. You can enter a disclaimer,
     instructions, ...etc.

   - You can change the navigation link text if you want. Normally leave that as
     it is.
